## Acerca del Proyecto

Este proyecto se creó como una librería pedagógica para enseñar a los estudiantes de pregrado de la facultad de Ingenieria y en particular del programa de Ingeniería en Sistemas e Informática de la Universidad de los Andes para usar y entender las estructuras de datos y sus algoritmos relacionados. Esto incluye estructuras tales como los arreglos, las listas, los mapas, los mapas ordenados, los árboles de búsqueda binarios (BST), árboles rojos negros (RBT) y los grafos.

**IMPORTANTE** Este es un trabajo en progreso. El proyecto se centra principalmente como una herramienta pedagógica para estudiantes universitarios y no está destinado a ser utilizado como una biblioteca completamente funcional.

[Volver al inicio](#disclib)

### Estructura

El proyecto consta de cuatro partes principales:
1. [DISClib](./DISClib) Carpeta principal con la implementación de la librería.
    1. [ADT](./DISClib/ADT) Carpeta con los Tipos de Datos Abstractos (ADT) más importantes implementados para la librería.
        1. [List](./DISClib/ADT/list.py) Script implementando las listas ADT en la librería, configurables por las estructuras Arreglos (Array List), Lista Sencillamente Encadenada (Single Linked List) y Listas Doblemente Encadenada (Double Linked List).
        1. [Queue](./DISClib/ADT/queue.py) Script que implementa el ADT Cola basada en el ADT Lista.
        1. [Stack](./DISClib/ADT/stack.py) Script que implementa el ADT Pila basada en el ADT lista.
        1. [Map](./DISClib/ADT/map.py) Script implementando el ADT Mapa utilizando Tablas Hash como estructuras de datos, configurables por método de manejo de colisiones a Sondeo Lineal (Linear Probing) y Encadenamiento Separado (Separate Chaining).
        1. [Ordered Map](./DISClib/ADT/orderedmap.py) Script que implementa los ADT de mapas ordenados utilizando los árboles binarios como estructuras de datos, configurables a Arboles Binarios de Búsqueda (binary search trees) o BST y Arboles Rojo-Negro (Red Black Tree) o RBT.
        1. [MinPQ](./DISClib/ADT/minpq.py) Script que implementa el ADT para la cola de prioridad mínima utilizando las estructuras de datos de Montículo (Heap).
        1. [Indexed MinPQ](./DISClib/ADT/indexminpq.py) Script que implementa el ADT para la cola de prioridad mínima indexada utilizando estructuras de datos de Montículo Indexado (Index Heap).
        1. [Graph](./DISClib/ADT/graph.py) Script que implementa el ADT Grafo utilizando listas de adyacencia como estructuras de datos.

    1. [Algorithms](./DISClib/Algorithms) Carpeta con los principales algoritmos implementados para procesar el contenido de los ADTs.
        1. [Graphs](./DISClib/Algorithms/Graphs) Carpeta con los algoritmos para procesar los ADT grafos.
            1. [DFS](./DISClib/Algorithms/Graphs/dfs.py) Script que implementa el algoritmo Búsqueda por Profundidad (Depth First Search) o DFS.
            1. [BFS](./DISClib/Algorithms/Graphs/bfs.py) Script que implementa el algoritmo de Búsqueda por lo Ancho (Breath First Search) o BFS.
            1. [DFO](./DISClib/Algorithms/Graphs/dfo.py) Script que implementa el algoritmo de ordenamiento topológico utilizando la búsqueda por profundidad topológica. También llamado Ordenamiento por profundidad (Depth First Order) o DFO.
            1. [Cycles](./DISClib/Algorithms/Graphs/cycles.py) Script que implementa el algoritmo detectar los ciclos en un grafo.
            1. [SCC](./DISClib/Algorithms/Graphs/scc.py) Script que implementa el algoritmo para detectar Componentes Fuertemente Conectados (SCC).
            1. [Prim](./DISClib/Algorithms/Graphs/prim.py) Script que implementa el algoritmo de Prim.
            1. [Dijkstra](./DISClib/Algorithms/Graphs/dijkstra.py) Script que implementa el algoritmo de Dijkstra.
            1. [Bellman-Ford](./DISClib/Algorithms/Graphs/bellmanford.py) Script que implementa el algoritmo Bellman-Ford.
        1. [Sorting](./DISClib/Algorithms/Sorting) Carpeta con las implementaciones de los algoritmos de ordenamiento utilizados en el ADT Lista.
            1. [Selection Sort](./DISClib/Algorithms/Sorting/selectionsort.py) Script que implementa el algoritmo de Ordenamiento por Selección (Selection Sort).
            1. [Insertion Sort](./DISClib/Algorithms/Sorting/insertionsort.py) Script que implementa el algoritmo de Ordenamiento por Inserción (Insertion Sort).
            1. [Shell Sort](./DISClib/Algorithms/Sorting/shellsort.py) Script que implementa el algoritmo de Ordenamiento de Shell (Shell Sort).
            1. [Merge Sort](./DISClib/Algorithms/Sorting/mergesort.py) Script que implementa el algoritmo de Ordenamiento por Unión (Merge Sort).
            1. [Quick Sort](./DISClib/Algorithms/Sorting/quicksort.py) Script que implementa el algoritmo de Ordenamiento Rápido (quick sort).
        1. [Trees](./DISClib/Algorithms/Trees) Carpeta con los algoritmos para administrar los árboles ADT.
            1. [Traversal](./DISClib/Algorithms/Trees/traversal.py) Script que implementa algoritmos transversales para los recorridos de árboles binarios, incluye los recorridos preorden, inorder y postorder.

    1. [Data Structures](./DISClib/DataStructures) Carpeta con las principales estructuras de datos implementadas para soportar los ADTs en la librería.
        1. [Array List](./DISClib/DataStructures/arraylist.py) Script que implementa el Arreglo (Array List) como estructura de datos.
        1. [Single Linked List](./DISClib/DataStructures/singlelinkedlist.py) Script que implementa la Lista Sencillamente Encadenada (Single Linked List) como estructura de datos.
        1. [Double Linked List](./DISClib/DataStructures/doublelinkedlist.py) Script que implementa la Lista Doblemente Encadenada (Double Linked List) como estructura de datos.
        1. [Heap](./DISClib/DataStructures/heap.py) Script que implementa Montículo (Heap) como estructura de datos.
        1. [Indexed Heap](./DISClib/DataStructures/indexheap.py) Script que implementa Montículo Indexado (Indexed Heap) como estructura de datos.
        1. [Indexed MinPQ Node](./DISClib/DataStructures/iminpqnode.py) Script que implementa la estructura de datos para el nodo MinPQ. También, es utilizado para apoyar el funcionamiento del Montículo Indexado (Indexed Heap).
        1. [Separate Chaining Hash Table](./DISClib/DataStructures/chaininghashtable.py) Script que implementa la Tabla de Hash por encadenamiento separado (Separate Chaining) como estructura de datos.
        1. [Linear Probing Hash Table](./DISClib/DataStructures/probehashtable.py) Script que implementa la Tabla de Hash Por Sondeo Lineal (Linear Probing) como estructura de datos.
        1. [Map Entry](./DISClib/DataStructures/mapentry.py) Script que implementa la estructura de datos de una entrada o valor de mapa para agregar a una tabla hash.
        1. [Binary Search Tree (BST)](./DISClib/DataStructures/bst.py) Script que implementa la estructura de datos del árbol de búsqueda binaria (BST) para el ADT de mapa ordenado.
        1. [BST Node](./DISClib/DataStructures/bstnode.py) Script que implementa el nodo de la estructura de datos BST y apoyar el funcionamiento del árbol de búsqueda binaria (BST).
        1. [Red Black Tree (RBT)](./DISClib/DataStructures/rbt.py) Script que implementa la estructura de datos del árbol rojo-negro (RBT) para el ADT de mapa ordenado.
        1. [RBT Node](./DISClib/DataStructures/rbtnode.py) Script que implementa el nodo de la estructura de datos RBT y apoyar el funcionamiento del árbol rojo-negro (RBT).
        1. [Adjacency List](./DISClib/DataStructures/adjlist.py) Script que implementa la estructura de datos de la Lista de Adyacencia para el ADT Grafo.
        1. [Edge](./DISClib/DataStructures/edge.py) Script que implementa la estructura de datos de Vértice o Nodo (Edge) de un grafo y apoya el funcionamiento del ADT grafo y su lista de adyacencia.

    1. [Utils](./DISClib/Utils) Carpeta con las funcionalidades genéricas principales que apoyan el funcionamiento de los ADTs en la librería.
        1. [Error Handling](./DISClib/Utils/error.py)  Script que implementa el control de errores para todas las funcionalidades de la librería.

1. [Test](./Test) Carpeta con las pruebas de la librería.
    1. [List](./Test/list) Scripts para evaluar los ADT Listas.
    1. [Queue](./Test/queue) Scripts para evaluar el ADT Cola.
    1. [Stack](./Test/stack) Scripts para evaluar el ADT Pila.
    1. [Map](./Test/map) Scripts para evaluar el ADT Mapa.
    1. [Ordered Map](./Test/orderedmap) Scripts para evaluar el ADT Mapa Ordenado.
    1. [Binary Search Tree](./Test/bst) Scripts para evaluar el ADT para el árbol de búsqueda binaria.
    1. [MinPQ](./Test/minpq) Scripts para evaluar el ADT de la cola de prioridad mínima.
    1. [Graph](./Test/graph) Scripts para evaluar el ADT Grafo.

**NOTA:** `DISClib` utiliza los scripts `config.py` para configurar la ruta de compilación de la librería y permitirle al intérprete Python encontrar la ruta relativa en cualquier condición del sistema operativo.

### Instalación

Por el momento, la librería **DISClib** está disponible como una dependencia local para su proyecto. Para instalarlo y usarlo debe seguir los siguientes pasos:

1. Cree una nueva carpeta de proyecto.
1. Clone el repositorio con el comando:

   ```sh
   git clone https://github.com/ISIS1225DEVS/ISIS1225-Lib.git
   ```

1. Mueva la carpeta **DISClib** a la carpeta del proyecto.
1. Cree un nuevo archivo Python en la carpeta del proyecto.
1. Importe los módulos ADT **DISClib** necesarios en su proyecto con las lineas de codigo:

   ```python
    from DISClib.ADT import list as lt
    from DISClib.ADT import map as mp
   ```

1. ¡Empieza a programar!
